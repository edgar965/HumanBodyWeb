🎭 Theatre.js — Quick Start Guide

Was ist Theatre.js?

Ein professionelles Tool für 3D-Animation und Motion Design. Perfekt für:

  • Kamera-Bewegungen (Dolly, Pan, Zoom)
  • Licht-Animation (Spotlights, Farbwechsel)
  • Character-Positionierung
  • Synchronisierte Animationen

🎬 Die Benutzeroberfläche

📋 Outline (links)

Liste aller animierbaren Objekte:

  • Camera
  • Spot Left
  • Spot Right
  • Back Light
  • Character (nach Import)

Tipp: Klick auf ein Objekt um es auszuwählen

⚙️ Details Panel (rechts)

Properties des ausgewählten Objekts:

  • Position (x, y, z)
  • Intensity (Lichter)
  • Color (RGBA)
  • FOV (Kamera)

Tipp: Werte mit Maus ziehen

🎞️ Timeline (unten)

Zeitleiste für Keyframe-Animation:

  • Playhead (roter Balken): Aktuelle Zeit
  • Play-Button (Leertaste): Animation abspielen
  • Keyframes: Automatisch erstellt beim Wert ändern
  • Sequenz-Länge: 10 Sekunden default (einstellbar)

🚀 Schnellstart: 30-Sekunden-Tutorial

  1. Öffne Theatre Seite
  2. File → Modell Laden → "FemaleWithHair" auswählen
  3. Im Outline links: "Camera" anklicken
  4. Im Details Panel rechts: FOV-Slider auf 20 ziehen (Zoom)
  5. Playhead in Timeline auf 5 Sekunden schieben
  6. FOV-Slider auf 50 ziehen (Zoom out) → Keyframe entsteht automatisch
  7. Leertaste drücken → Animation startet!

💡 Warum sehe ich keine Lichter?

Die Lichter sind unsichtbare Objekte (wie in echten 3D-Apps). Du siehst nur ihre Wirkung auf den Character.

Tipp: Wähle "Spot Left" im Outline → ändere "intensity" auf 15 → siehst du mehr Licht?

📚 Wichtige Shortcuts

Taste Aktion
Leertaste Play/Pause Animation
Mausrad Canvas: Zoom in/out
Linke Maustaste + Ziehen Canvas: Kamera drehen (OrbitControls)
Rechte Maustaste + Ziehen Canvas: Kamera verschieben

🎯 Typische Workflows

🎥 Kamera-Fahrt erstellen

1. Camera auswählen
2. Playhead auf 0:00
3. Position.z = 8 (weit weg)
4. Playhead auf 5:00
5. Position.z = 3 (nah ran)
→ Kamera fährt langsam auf Character zu

💡 Licht-Farbwechsel

1. "Spot Left" auswählen
2. Playhead auf 0:00
3. Color = Warmes Weiß (r:1, g:0.9, b:0.8)
4. Playhead auf 5:00
5. Color = Kaltes Blau (r:0.6, g:0.7, b:1)
→ Licht wechselt langsam die Farbe

🎭 Character + Animation

1. File → Modell Laden → FemaleWithHair
2. File → Animation Laden → Mixamo → Ballet
→ Character tanzt automatisch
3. Character im Outline auswählen
4. Position/Rotation animieren (wie Kamera)
→ Character bewegt sich WÄHREND er tanzt

⚠️ Häufige Probleme

Character liegt auf dem Boden!
→ Hard Refresh im Browser (Ctrl+Shift+R) — Bug ist bereits gefixt
Modell laden funktioniert nicht!
→ Hard Refresh im Browser (Ctrl+Shift+R) — Meta-Slider-Bug ist gefixt
Ich sehe nur Schwarz!
→ Lichter sind zu dunkel. Wähle "Spot Left" → Intensity auf 15 hochdrehen

🎬 Video Export

Wenn du mit deiner Animation zufrieden bist:

  1. Playhead auf 0:00 zurücksetzen
  2. Export Video Button klicken (oben rechts)
  3. Leertaste drücken → Animation läuft ab
  4. Wenn fertig: Nochmal Stop Recording klicken
  5. Browser lädt theatre-export.webm herunter
