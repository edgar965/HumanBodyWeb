Theatre.js — Vollständige Anleitung

Kamera

Kamera bewegen (Maus)

Rechtsklick + Ziehen Kamera verschieben (Pan)
Linksklick + Ziehen Kamera drehen (Orbit)
Scroll-Rad Zoom rein / raus

Kamera-Presets (Schnellwechsel)

Menü "Settings" oben → Preset auswählen:

Preset Kamera FOV Stil
Ballet Stage Mittel (z=5) 35 Theatral, warm
Studio Bright Nah (z=4) 40 Hell, gleichmäßig
Cinematic Moody Seitlich (x=2) 28 Film noir, Tele-Objektiv
Fashion Show Weit (z=6) 50 Runway, Weitwinkel
Sunset Warm Seitlich (x=-1) 42 Goldene Stunde

Kamera-Bewegung animieren (Keyframes)

  1. Rechtes Panel → Tab "Keyframes"
  2. Objekt-Dropdown: "Camera" auswählen
  3. Kamera mit Maus an die Startposition bringen
  4. Timeline-Slider auf 0 Sekunden
  5. "Add Keyframe" klicken → Position wird gespeichert
  6. Kamera an die Zielposition bewegen
  7. Timeline-Slider auf z.B. 3 Sekunden
  8. "Add Keyframe" klicken
  9. "Play" drücken → Kamera gleitet sanft von A nach B

Kamerawechsel: Totale → Nahaufnahme

Harter Schnitt (Cut)

Zwei Keyframes direkt nebeneinander setzen:

Keyframe 1:  Zeit = 5.00s  |  Position z = 6   |  FOV = 50  (Totale)
Keyframe 2:  Zeit = 5.01s  |  Position z = 1.5 |  FOV = 28  (Nahaufnahme Gesicht)

→ Sofortiger Sprung zur Nahaufnahme (wie Filmschnitt)

Sanfte Kamerafahrt (Dolly)

Zwei Keyframes mit Zeitabstand:

Keyframe 1:  Zeit = 3.0s  |  Position z = 6   |  FOV = 50  (Totale)
Keyframe 2:  Zeit = 6.0s  |  Position z = 1.5 |  FOV = 28  (Nahaufnahme)

→ Kamera fährt 3 Sekunden lang sanft zum Gesicht

Mehrere Kameraperspektiven hintereinander

0.0s  Totale von vorne       (z=5, y=1.6,  FOV=50)
3.0s  Seitenansicht           (x=4, z=0,    FOV=40)
3.01s Nahaufnahme Gesicht     (z=1.5, y=1.7, FOV=28)
6.0s  Halbtotale von hinten   (z=-4, y=1.5, FOV=35)
9.0s  Zurück zur Totale       (z=5, y=1.6,  FOV=50)

Lichter

Verfügbare Lichter

Name Standard-Position Funktion
Spot Left Links oben (-3, 6, 3) Hauptlicht links
Spot Right Rechts oben (3, 6, 3) Hauptlicht rechts
Back Light Hinten oben (0, 5, -4) Gegenlicht / Rim Light
Ambient — Grundhelligkeit (immer an)

Lichter verschieben

  1. Toolbar: "Translate"-Modus muss aktiv sein (Standard)
  2. Ein Licht-Icon (Kegel-Symbol) im Viewport anklicken
  3. Achsen-Pfeile (rot/grün/blau) erscheinen
  4. Pfeil ziehen um das Licht zu verschieben

Alternativ: Rechtes Panel → Tab "Eigenschaften" → Position X/Y/Z direkt eingeben.

Toolbar-Button "Licht-Icons" schaltet die Kegel-Anzeige ein/aus.

Licht-Einstellungen ändern

Intensität Properties-Tab → Intensity-Slider (0 = aus, 100 = max)
Farbe Properties-Tab → R/G/B Werte (0.0 – 1.0)
Ein/Aus Intensity = 0 (aus) oder hochdrehen (an)

Licht-Bewegung animieren

Funktioniert genau wie bei der Kamera:

  1. Tab "Keyframes" → Objekt z.B. "Spot Left" auswählen
  2. Licht an Startposition ziehen, Intensity einstellen
  3. Timeline auf 0s → "Add Keyframe"
  4. Licht an neue Position ziehen, Intensity ändern
  5. Timeline auf 4s → "Add Keyframe"
  6. Play → Licht gleitet interpoliert zwischen den Positionen

Licht ein-/ausblenden (Fade)

Einblenden:
Keyframe 1:  Zeit = 2.0s  |  Intensity = 0   (aus)
Keyframe 2:  Zeit = 3.0s  |  Intensity = 60  (an)
→ Licht blendet in 1 Sekunde sanft ein

Ausblenden:
Keyframe 1:  Zeit = 7.0s  |  Intensity = 60  (an)
Keyframe 2:  Zeit = 8.0s  |  Intensity = 0   (aus)
→ Licht blendet in 1 Sekunde sanft aus

Licht-Farbwechsel animieren

Im Theatre.js Studio (Profi-Timeline):

1. "Spot Left" im Studio-Outline auswählen
2. Playhead auf 0:00
3. Color → r=1.0, g=0.9, b=0.8  (warmes Weiß)
4. Playhead auf 5:00
5. Color → r=0.6, g=0.7, b=1.0  (kaltes Blau)
→ Licht wechselt langsam die Farbe von warm nach kalt

Licht-Bewegung zwischen 2 Punkten interpolieren

Beispiel: Spot Left wandert von links nach rechts

Keyframe 1:  Zeit = 0s  |  Position x=-5, y=4, z=3   (links)
Keyframe 2:  Zeit = 5s  |  Position x=+5, y=4, z=3   (rechts)

→ Licht gleitet in 5 Sekunden von links nach rechts

Beispiel: Back Light kreist (4 Keyframes)

0s:  Position x=0,  z=-4  (hinten)
2s:  Position x=4,  z=0   (rechts)
4s:  Position x=0,  z=4   (vorne)
6s:  Position x=-4, z=0   (links)
8s:  Position x=0,  z=-4  (zurück hinten)

→ Licht kreist um den Character

Komplette Szene aufbauen

Beispiel: Dramatische Tanzszene

1. Setup
   Settings → "Cinematic Moody" (dunkle Stimmung)
   Modelle-Tab → "FemaleWithHair" laden
   Animationen-Tab → Tanz-BVH laden

2. Kamera-Keyframes
   0.0s   Totale von vorne        (z=5,  FOV=50)
   3.0s   Seitenansicht            (x=3, z=2, FOV=40)
   3.01s  Gesicht Nahaufnahme      (z=1.5, y=1.7, FOV=25)
   6.0s   Halbtotale schräg        (x=-2, z=3, FOV=35)
   9.0s   Zurück zur Totale        (z=5, FOV=50)

3. Licht-Keyframes (Spot Left)
   0.0s   Intensity=15, Position links  (x=-4)
   3.0s   Intensity=60, Position Mitte  (x=0)    ← Licht folgt Kamera-Cut
   6.0s   Intensity=15, Position rechts (x=4)
   9.0s   Intensity=15, zurück links    (x=-4)

4. Licht-Keyframes (Back Light)
   0.0s   Intensity=0   (aus)
   3.0s   Intensity=40  (einblenden für Nahaufnahme)
   6.0s   Intensity=0   (wieder aus)

5. Play → Kontrolle → Export MP4

Theatre.js Studio (Profi-Timeline)

Studio ein-/ausschalten

Toolbar-Button "Studio" → Theatre.js Studio-Overlay erscheint am unteren Rand.

Das Studio bietet feinere Kontrolle als das Keyframes-Panel:

  • Keyframe-Diamonds direkt auf der Timeline verschieben
  • Einzelne Tracks (Position X, Y, Z, Intensity, Color) separat bearbeiten
  • Easing/Interpolation zwischen Keyframes visuell sehen
  • Werte per Maus-Drag anpassen (im Detail Panel rechts)
Tipp: Im Studio entstehen Keyframes automatisch wenn du einen Wert änderst während der Playhead an einer bestimmten Zeitposition steht.

Alle Keyframes löschen

  • Toolbar-Button (Radierer) klicken → Bestätigen → Seite wird neu geladen ohne Keyframes
  • Alternativ: Rechtes Panel → Tab "Keyframes" → "Clear"-Button unten

Einzelne Keyframes löschen

  • Rechtes Panel → Tab "Keyframes" → -Icon neben dem jeweiligen Keyframe klicken

Neue Keyframes setzen

  • Rechtes Panel → Tab "Keyframes"
  • Objekt auswählen (Camera, Spot Left, etc.)
  • Timeline-Slider auf die gewünschte Zeit ziehen
  • Kamera/Licht im Viewport positionieren
  • "Add Keyframe at Current Time" klicken
Tipp: Im Studio-Panel (Profi-Timeline unten) entstehen Keyframes auch automatisch wenn du einen Wert im Detail-Panel rechts änderst.

Video Export

  1. Animation fertig aufgebaut (Kamera + Lichter + BVH)
  2. Menü "Datei" → "Export MP4"
  3. Einstellungen: Auflösung (720p–4K), FPS (24/30/60), Qualität
  4. Recording starten → Animation spielt ab
  5. Browser lädt die fertige Datei herunter

Formate: MP4 (via Server/ffmpeg) oder WebM (direkt im Browser)

Häufige Probleme

Character liegt auf dem Boden!
→ Hard Refresh: Ctrl+Shift+R
Alles ist schwarz!
→ Lichter zu dunkel. "Spot Left" auswählen → Intensity auf 60 hochdrehen.
Keyframes tun nichts!
→ Prüfe ob das richtige Objekt (Camera/Spot Left/etc.) im Dropdown ausgewählt ist.
Kamera springt beim Play!
→ Theatre.js übernimmt die Kamera während der Wiedergabe. Orbit-Steuerung geht nur im Pause-Modus.
