Pipelines

OpenPose (CUDA GPU): More accurate body pose detection. Requires NVIDIA GPU.

Processing Steps

  1. Video → 2D joint detection (MediaPipe or OpenPose)
  2. 2D Joints CSV → MocapNET Neural Network
  3. MocapNET → BVH file (CMU Skeleton, 164 joints)
