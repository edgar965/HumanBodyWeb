Hilfe — Animationen

Animationen verwalten

Die Animationsseite zeigt alle BVH-Dateien gruppiert nach Ordnern. Klicke auf eine Animation um sie abzuspielen.

Toolbar

Neuer Ordner — Erstellt einen neuen Kategorie-Ordner
Umbenennen — Benennt die ausgewaehlte Animation um (auch: F2)
Verschieben — Verschiebt die Animation in einen anderen Ordner
Loeschen — Loescht die ausgewaehlte Animation (auch: Del)
Aktualisieren — Laedt die Animationsliste neu

Kontextmenue (Rechtsklick)

Auf eine Animation:

  • Abspielen — Spielt die Animation ab
  • Umbenennen — Aendert den Dateinamen
  • Verschieben nach... — Verschiebt in einen anderen Ordner (Name eingeben)
  • Loeschen — Entfernt die BVH-Datei (mit Bestaetigung)

Auf einen Ordner:

  • Ordner umbenennen — Aendert den Ordnernamen
  • Neuer Unterordner — Erstellt einen neuen Ordner
  • Ordner loeschen — Entfernt einen leeren Ordner

Tastenkuerzel

F2Ausgewaehlte Animation umbenennen
DelAusgewaehlte Animation loeschen
RechtsklickKontextmenue fuer Datei oder Ordner

Wiedergabe

  • Play/Pause — Startet oder pausiert die Animation
  • Stop — Setzt die Animation auf Frame 0 zurueck
  • Speed-Slider — Aendert die Abspielgeschwindigkeit (0.1x bis 3.0x)
  • Timeline-Slider — Scrubbt durch die Animation

Hinweise

  • Ordner koennen nur geloescht werden wenn sie leer sind
  • Beim Umbenennen/Verschieben wird auch die Retarget-Cache-Datei (.json) mitverschoben
  • Alle Aenderungen werden sofort auf der Festplatte ausgefuehrt
